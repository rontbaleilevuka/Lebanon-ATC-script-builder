# Lebanon ATC Script Builder

A standalone web-based ATC simulation script builder for Lebanon ATC operations. No installation required — opens directly in any browser on computer or iPad.

**[User Guide & Troubleshooting](https://rontbaleilevuka.github.io/atc-script-builder-lebanon/ATC_Script_Builder_Guide_Lebanon.html)**

---

## Features
- Colour-coded ATC scripts matching the standard format
- IFR and VFR departure 3-call system with pushback and same-call options
- All airport fields are free text — type any ICAO code directly
- Aircraft type dropdown (194 sim aircraft) and separate transponder field
- Frequency management — toggle on TWR / GND / DEL / APP / DEP / INFO / APRON
- Save scripts as `.json` and load them back instantly
- Export to PDF for Apple Pencil annotation in GoodNotes or Notability
- ATIS and QNH footer reference on every script
- Edit individual entries without starting from scratch
- Keyboard navigation for fast data entry
- Works fully offline once loaded
